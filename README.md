# ComplyAI 🛡️

### The Authoritative Curator for Global AI Compliance

**ComplyAI** is a production-ready web application designed to help product teams, legal departments, and AI developers navigate the complex landscape of global AI regulations. Map your product's "AI DNA" and receive a real-time compliance score and jurisdictional roadmap.

---

## 🚀 Key Features

- **Interactive Feature Map:** Define your product's AI capabilities (Facial Recognition, GenAI, Autonomous Agents, etc.) and see their risk levels (Low, Medium, High).
- **Global Jurisdictional Selection:** Choose your target regions (EU, US, Singapore, India, China) and see active regulatory frameworks (EU AI Act, GDPR, NIST, etc.).
- **Real-time Compliance Scoring:** A dynamic score that updates as you toggle features and regions, providing immediate feedback on your regulatory risk.
- **Multi-Region Compliance Reports:** Generate detailed, region-specific reports with executive summaries and required actions.
- **Policy Library:** Browse a curated database of 400+ global AI regulations and frameworks with regional associations.
- **Trust-Tech Design:** A clean, editorial-style UI built for clarity, precision, and professional use.

---

## 🛠️ Tech Stack

- **Frontend:** React 18+, TypeScript, Vite
- **Styling:** Tailwind CSS (Utility-first styling)
- **Animations:** Framer Motion (Smooth transitions and interactive elements)
- **Icons:** Lucide React (Consistent, high-quality iconography)
- **Deployment:** Optimized for Cloud Run and GitHub Pages

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/arjun-basu001/ComplyAI.git
   cd ComplyAI
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

---

## 🌍 Deployment

This project is configured for easy deployment to **GitHub Pages**. The landing page is located in the `docs/` directory.

To deploy:
1. Go to your GitHub repository settings.
2. Navigate to the **Pages** section.
3. Select **Deploy from a branch**.
4. Choose the `main` branch and the `/docs` folder.
5. Save and wait for the deployment to complete.
